## 4. Boolean Search Operators

### Advanced Search Techniques:

**OR Operator (Broaden Search):**
```
"Commercial Real Estate Broker" OR "CRE Broker" OR "Tenant Representative"
```

**AND Operator (Narrow Search):**
```
"Property Manager" AND "Commercial" AND "Senior"
```

**NOT Operator (Exclude Terms):**
```
"Real Estate" NOT "Residential" NOT "Residential Broker"
```

**Quotes for Exact Phrases:**
```
"Corporate Real Estate Director"
"Commercial Property Manager"
```

**Parentheses for Grouping:**
```
("Property Manager" OR "Asset Manager") AND ("Commercial" OR "CRE")
```

### Example Complex Query:
```
("Commercial Real Estate" OR "CRE") AND ("Broker" OR "Manager" OR "Director") NOT "Residential"
```

## 17. Prospecting Workflow

### Daily Prospecting Routine:

**Morning (30 minutes):**
1. Check saved search alerts (new matches)
2. Review 10-15 new prospects
3. Send 5-10 connection requests (personalized)
4. Follow up on pending requests

**Afternoon (30 minutes):**
1. Engage with prospects' content (likes, comments)
2. Research 5-10 companies
3. Identify decision-makers at target accounts
4. Update CRM/lead list

**Weekly:**
1. Review and refine saved searches
2. Analyze what's working (response rates)
3. Adjust targeting based on results
4. Create new saved searches for new segments

## 18. Search Refinement Tips

### How to Refine Searches:

**Too Many Results (>1000):**
- Add more specific job title keywords
- Narrow geographic location
- Add company size filter
- Exclude more industries
- Add more "AND" conditions

**Too Few Results (<50):**
- Remove some filters
- Use "OR" instead of "AND"
- Expand geographic area
- Remove company size restrictions
- Use broader job title keywords

**Wrong Types of People:**
- Add exclusion keywords (NOT operators)
- Refine industry filters
- Be more specific with job titles
- Check company descriptions

## 20. Best Practices

### Do's:
✅ **Personalize connection requests** (mention something specific)
✅ **Research before reaching out** (check their profile, recent posts)
✅ **Start with value** (share insight, not pitch)
✅ **Be consistent** (daily prospecting routine)
✅ **Track what works** (measure response rates by segment)
✅ **Follow up** (if no response, try again in 1-2 weeks)
✅ **Engage before pitching** (like/comment on their posts first)

### Don'ts:
❌ **Don't spam** (mass connection requests without personalization)
❌ **Don't pitch immediately** (build relationship first)
❌ **Don't ignore filters** (target the right people)
❌ **Don't give up too quickly** (follow up 2-3 times)
❌ **Don't use generic messages** (personalize every outreach)
❌ **Don't ignore engagement** (respond to comments, messages)
❌ **Don't forget to qualify** (ensure they're a good fit before deep dive)
